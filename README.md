# Projekty Python – kružnice a signály

## 1. Kružnice

Tento projekt zistí, či sa dve kružnice pretínajú, koľko majú prienikov a vykreslí ich do grafu.

### Súbory
- `circle_stats.py` – výpočty (prienik, súčet polomerov, vzdialenosť) + testy
- `circles_intersection_draw.py` – vykreslenie kružníc
- `circle_intersection.py` – hlavný skript
